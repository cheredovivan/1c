Пересечение правил расчета резервов под обесценение

Отчет предназначен для выявления ТМЦ, по которым одновременно действуют несколько правил формирования резервов под обесценение запасов.

Варианты формирования отчета:

  • Выводить все - выводятся все ТМЦ по организации, для которых есть действующие правила и остатки на период отчета;
  • Все пересечения - выводятся ТМЦ по организации, для которых есть более одного действующего правила и остатки на период отчета;
  • Только конфликты - выводятся ТМЦ по организации, для которых есть более одного действующего правила с одинаковым приоритетом и остатки на период отчета.
