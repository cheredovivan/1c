Применение правила расчета резервов под обесценение

Отчет предназначен для отображения остатков ТМЦ, попадающих под действие правила формирования резервов под обесценение запасов. Формируется список ТМЦ, данные по их себестоимости, чистой стоимости продажи, а также отклонении между ними. Возможно формирование в валюте регламентированного и управленческого учета.

Варианты формирования отчета:

  • Все остатки по правилу - выводятся все ТМЦ, удовлетворяющие условиям правила, по которым есть остатки на период отчета;
  • Только с ценой - выводятся ТМЦ, удовлетворяющие условиям правила, по которым есть остатки и установлена цена по виду цен, указанному в правиле, на период отчета;
  • Только без цены - выводятся ТМЦ, удовлетворяющие условиям правила, по которым есть остатки и не установлена цена по виду цен, указанному в правиле, на период отчета.
