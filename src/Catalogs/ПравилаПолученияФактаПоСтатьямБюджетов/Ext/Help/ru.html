Правила получения фактических данных по статьям бюджетов

Правила предназначены для настройки получения фактических данных по Статьям бюджетов.
Сбор фактических данных производится либо в бюджетных отчетах, предполагающих вывод фактических данных, либо при планировании, при заполнении планов на основе фактических данных, в документеЭкземпляра бюджета.

Правила получения фактических данных бюджетирования настраиваются для бюджетных статей и показателей. Для статей бюджетов правила могут быть настроены как для собственно получения фактических оборотов по статье, в этом случае указывается предопределенный сценарий Фактические данные, так и для указания правил для контроля лимитов расходования ДС, для этого применяется сценарий Исполнение бюджета.

Раздел источника данных определяет, какие источники будут использоваться для получения фактических данных.

Возможные разделы учета:

  • Оперативный учет, то источниками данных являются обороты по Хозяйственным операциям.
  • Регламентированный учет и Международный учет, источниками данных являются обороты по счетам соответствующего плана счетов.
  • Произвольные данные позволяет настроить произвольную схему получения фактических данных.
