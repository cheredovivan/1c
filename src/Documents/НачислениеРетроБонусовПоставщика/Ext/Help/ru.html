Начисление ретро-бонусов поставщика

Документ предназначен для расчёта и фиксации результата начисления сумм ретро-бонусов по итогам выполнения условий получения ретро-бонусов, зафиксированных в документе Условия ретро-бонусов поставщика.

По команде Заполнить по условиям табличную часть документа можно заполнить суммами рассчитанного бонуса по данным закупок, продаж или остатков товаров в соответствии с условиями ретро-бонуса.
