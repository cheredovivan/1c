Производственная операция

Документ предназначен для управления выполнением операций этапа производства. Описывает выполнение одной операции технологического маршрута на заданное количество. Формируется для этапов в состоянии "Начат" из рабочего места Выполнение операций из пула запланированных к выполнению операций, если в производственном подразделении не включено использование сменных заданий. При использовании сменных заданий операции формируются в рабочем месте формирование сменных заданий путем назначения операции на смену.
